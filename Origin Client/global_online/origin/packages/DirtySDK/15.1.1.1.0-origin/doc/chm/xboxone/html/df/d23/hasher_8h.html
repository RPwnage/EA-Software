Detailed Description

This module implements some simple hash table functions. Both numeric and string based versions are provided. This module does not do key or data storage allocation– it is up to the caller to keep the keys and data in memory that persists for the life of the hash table. The string hash functions do not consider case nor spaces when comparing.
