file  protohttp.h [code]
 This module implements an HTTP client that can perform basic transactions (get/put) with an HTTP server. It conforms to but does not fully implement the 1.1 HTTP spec (http://www.w3.org/Protocols/rfc2616/rfc2616.html), and allows for secure HTTP transactions as well as insecure transactions.

A client application can submit rapid-fire http requests and ProtoHttpManager will distribute them efficiently across the ref pool internally, queuing them for efficient use of keep-alive and pipelining requests where possible.
