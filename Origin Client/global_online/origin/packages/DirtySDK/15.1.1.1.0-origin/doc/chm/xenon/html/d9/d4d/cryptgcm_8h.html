Detailed Description

An implementation of the GCM-128 and GCM-256 cipher, based on the NIST standard, intended for use in implementation of TLS1.1 GCM cipher suites.

This implementation uses Shoup's method utilizing 4-bit tables as described in the GCM specifications. While not particularly fast, table generation is quick and memory usage required small. This implementation is restricted to a feature set suitable for implementation of TLS1.1 GCM cipher suites.
