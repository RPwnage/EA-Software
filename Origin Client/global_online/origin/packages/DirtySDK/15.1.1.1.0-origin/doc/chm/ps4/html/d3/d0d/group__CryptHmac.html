Implements HMAC as defined in http://tools.ietf.org/html/rfc4346#section-5. This version allows multiple buffers to be specified in one call, which is useful when the input to be hashed is not in a single contiguous buffer.

Parameters
*pBuffer- [out] storage for generated MAC
iBufLen- size of output buffer (may be smaller than hash size for truncated HMAC)
*pMessageList-list of messages to hash
iNumMessages- number of messages to hash
*pKey- seed
iKeyLen- seed length
eHashType- hash type
Returns
int32_t - negative=error, else success
Version
01/14/2013 (jbrookes)
