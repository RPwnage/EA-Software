Detailed Description

This module implements a WebSocket client as documented in RFC 6455 (http://tools.ietf.org/html/rfc6455). Note that implementation details may vary when a platform-specific implementation is required. The API as currently implemented offers stream-like operations; message-based operations are currently not supported.
