Resolve an address.

Parameters
*ref- module ref
*addr- resolve address
*buf- target buffer
len- target length (min 64 bytes)
div- divider char
Returns
int32_t - <0=error, 0=complete (COMM_NOERROR), >0=in progress (COMM_PENDING)
Note
Target list is always double null terminated allowing null to be used as the divider character if desired. when COMM_PENDING is returned, target buffer is set to "~" until completion.
