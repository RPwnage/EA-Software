int32_t FriendApiPrivBlockUser ( FriendApiRefT *  pFriendApiRef,
uint32_t  uUserIndex,
void *  pUserId 
)

Initiates a block user operation. After calling this function, if the return code is "success", the caller needs to wait for the FRIENDAPI_EVENT_BLOCK_USER_RESULT completion event.

Parameters
*pFriendApiRef- module reference
uUserIndex- local user index
*pUserId- user to be blocked (pointer to SceNpId)
Returns
int32_t - 0 for success, negative for failure
Note
Experimentation showed that, in a scenario where the specified user is already a friend, sceNpBasicAddBlockListEntry() returns 0 for success, but later in time the NP Basic handler is invoked by the system with: iEvent=SCE_NP_BASIC_EVENT_ADD_BLOCKLIST_RESULT/11 iRetCode=SCE_NP_BASIC_ERROR_BLOCKLIST_ADD_FAILED/0x8002a675
Version
02/24/2011 (mclouatre)
