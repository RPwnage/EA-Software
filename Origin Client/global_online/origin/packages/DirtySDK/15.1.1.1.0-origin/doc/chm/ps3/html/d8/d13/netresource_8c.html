Detailed Description

Resource downloader/caching utility.

Note
The network resource manager provides game titles with a simple method to download arbitrary resources (binary objects of any underlying type) via a simple type/identifier combination. Each game will have its own unique type/identifier mapping to avoid conflicts with other titles.
