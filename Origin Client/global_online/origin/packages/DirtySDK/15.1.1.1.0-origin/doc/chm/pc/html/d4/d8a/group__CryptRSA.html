Init RSA state.

Parameters
*pState- module state
*pModulus- crypto modulus
iModSize- size of modulus
*pExponent- crypto exponent
iExpSize- size of exponent
Returns
int32_t - zero for success, negative for error
Note
This module supports a max modulus size of 4096 (iModSize=512) and requires a minimum size of 1024 (iModSize=64). The exponent must be between 1 and 512 bytes in size, inclusive.
