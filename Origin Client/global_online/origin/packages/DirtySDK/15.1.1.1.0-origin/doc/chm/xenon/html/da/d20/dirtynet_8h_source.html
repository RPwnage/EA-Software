174 /*
175 Macros used to write into a sockaddr structure take into account the fact that the sa_data field is defined
176 as an array of signed char on Windows and an array of unsigned char on other platforms. Doing this is required
177 to avoid C4365 warning occurring when these macros are used in .cpp files. (The same warning would not occur
178 when using these macros in .c files. The windows compiler (cl.exe) only warns about C4365 with .cpp files.
