5  \Description
6  This module is a from-scratch ARC4 implementation designed to avoid
7  any intellectual property complications. The ARC4 stream cipher is
8  known to produce output that is compatible with the RC4 stream cipher.
9 
10  \Notes
11  This algorithm from this cypher was taken from the web site: ciphersaber.gurus.com
12  It is based on the RC4 compatible algorithm that was published in the 2nd ed of
13  the book Applied Cryptography by Bruce Schneier. This is a private-key stream
14  cipher which means that some other secure way of exchanging cipher keys prior
15  to algorithm use is required. Its strength is directly related to the key exchange
16  algorithm strengh. In operation, each individual stream message must use a unique
17  key. This is handled by appending on 10 byte random value onto the private key.
18  This 10-byte data can be sent by public means to the receptor (or included at the
19  start of a file or whatever). When the private key is combined with this public
20  key, it essentially puts the cypher into a random starting state (it is like
21  using a message digest routine to generate a random hash for password comparison).
22  The code below was written from scratch using only a textual algorithm description.
23
