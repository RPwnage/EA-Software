 Captures DirtySDK debug output and posts it to a webserver where the output can be retrieved. This is useful when debugging on a system with no debugging capability or in a "clean room" environment, for example. Two basic mechanisms are employed; the first is a NetPrint debug hook to capture all debug output, the second a stand-alone WebOfferPrintf() function that can be used more selectively.
