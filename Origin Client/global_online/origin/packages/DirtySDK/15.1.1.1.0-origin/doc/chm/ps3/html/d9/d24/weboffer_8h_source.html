161 /*!
162  Games that support Greek and/or Asian languages encode each
163  user-visible character using UTF-8 which results in a single
164  character being represented by 1-4 bytes. These games therefore
165  need larger buffers to hold strings.
166 
167  The following macro is used to convert the suggested display
168  length to a storage size. If space is at a premium and the game
169  is English only, the calculation can be reduced to simply equate
170  the two sizes. Otherwise, this formula should be adjusted to take
171  into account the average display size and average storage size of
172  the strings to be used.
173 */
174 #define WEBOFFER_DispToMemSize(nDispChars) ((nDispChars * 100) / 50)
