1 /*H*************************************************************************************/
2 /*!
3  \File xmlformat.h
4 
5  \Description
6  This module formats simple Xml, in a linear fashion, using a character buffer
7  that the client provides.
8 
9  \Notes
10  \verbatim
11  Since this is a simple linear buffer write, without support for a DOM, the following
12  rules apply:
13  - You cannot set attributes or element values if an element (tag) is not started.
14  - If you start an element, (tag), you have to set the attributes before any
15  child elements are added.
16  - The XmlElemAddXXX calls will create a complete element, complete with start
17  and end tags.
18  - XmlInit MUST be called on the client buffer, before any api functions are
19  called.
20  - Once XML output is complete, the client should call XmlFinish(). There is
21  some hidden data in the buffer which must be cleared prior to output. Client
22  should account for 24 extra bytes in buffer for use by Xml API.
23 
24  References:
25  1) XML Core: http://www.w3.org/XML/Core/
26  2) XML 1.0 4th edition specifications: http://www.w3.org/TR/xml/
27  \endverbatim
28

32  \Version 01/30/2004 (jbertrand) First Version
33 */
