Detailed Description

High-level module designed to create and manage a pool of ProtoHttp refs. A client application can submit rapid-fire http requests and ProtoHttpManager will distribute them efficiently across the ref pool internally, queuing them for efficient use of keep-alive and pipelining requests where possible.
