Detailed Description

This module implements High Level Buddy, an application level interface to the Buddy server.

Attention
Copyright (c) Electronic Arts 2003. ALL RIGHTS RESERVED.
Note
Depends on and encapsulates, the lower level dirty sock buddy api. (buddyapi.h and .c modules). Clients should ONLY use hlbudapi.c and NOT buddyapi.c
