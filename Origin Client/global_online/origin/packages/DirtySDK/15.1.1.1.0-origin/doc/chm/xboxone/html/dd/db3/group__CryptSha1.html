DIRTYCODE_API void CryptSha1Final ( CryptSha1T *  pSha1,
void *  pBuffer,
uint32_t  uLength 
)

Generate the final hash from the SHA1 state.

Parameters
*pSha1- the SHA1 state
*pBuffer- where the hash should be written
uLength- the number of bytes to write, [0..CRYPTSHA1_HASHSIZE]
Returns
none
Note
Usually callers want the whole hash and so uOutputLen would be CRYPTSHA1_HASHSIZE. However, if only a partial hash is needed then any value up to CRYPTSHA1_HASHSIZE can be used. Any value greater than CRYPTSHA1_HASHSIZE is silently truncated to CRYPTSHA1_HASHSIZE.

CryptSha1Final is not idempotent. It could easily be made so but callers typically tend not to need it so it is not supported.
