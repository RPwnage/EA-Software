Detailed Description

Resource downloader/caching utility.

Note
As several GDDs now require the use of dynamic downloaded images, some unified method of downloading these resources is required. The network resource manager would provide game titles with a simple method to download arbitrary resources (binary objects of any underlying type) via a simple type/identifier combination. Each game will have its own unique type/identifier mapping to avoid conflicts with other titles.
