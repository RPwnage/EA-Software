Detailed Description

ProtoPingManager is a high-level interface to ProtoPing. It provides both metering and and LRU caching of ping requests, so as to enable high-level applications to issue high volume ping requests without worrying about over- burdening the EE<->IOP interface or creating network-unfriendly ping storms.
