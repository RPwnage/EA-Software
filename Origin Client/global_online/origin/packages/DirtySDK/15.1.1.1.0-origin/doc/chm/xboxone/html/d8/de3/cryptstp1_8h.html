Detailed Description

Implements STP1 (session ticket protocol version one) for encrypting session streams via shared secret encoded tickets. RC4 is used for encryption and MD5 for data verification. Original protocol design by Stephen Bevan and module packaging by Greg Schaefer.

Module derived from sbevan encryption code originally in SessCrypt.cpp, in Server.cpp, ProtoAries.cpp, Dir-Main.cpp and LobbyAPI.c
