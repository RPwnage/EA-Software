Detailed Description

Implements JFIF image decoding to an output 32bit buffer. This decoder does not support progressive, lossless, or arithmetic options.

Note
References: [1] http://www.w3.org/Graphics/JPEG/jfif3.pdf [2] http://www.w3.org/Graphics/JPEG/itu-t81.pdf [3] http://class.ece.iastate.edu/ee528/Reading%20material/JPEG_File_Format.pdf [4] http://www.bsdg.org/SWAG/GRAPHICS/0143.PAS.html
