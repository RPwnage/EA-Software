Check one or more privileges against the system or the first party service for an authorotative determination of the current privilege status. No UI is shown when using this function.

Parameters
*pRef- the module reference
iUserIndex- user index
*pPrivileges- array of privileges
iPrivilegeCount- the numnber of privileges pointed to by pPrivileges
*pHint- (optional, out) if provided, pHint will be filled with a non-authoratative result for the privilege check. Unless the return value is zero (0), in which case, this value can be considered autorotative.
Returns
int32_t - <0 = error, 0 = success, use *pHint for the result, >0 = the request id to pass into PrivilegeApiCheckResult()

Check one or more privileges against the system or the first party service for an authorotative determination of the current privilege status. This function will show appropriate error messages or commerce dialog to upsell product/privileges to the user.

Parameters
*pRef- the module reference
iUserIndex- user index
*pPrivileges- array of privileges
iPrivilegeCount- the numnber of privileges pointed to by pPrivileges
*pHint- (optional, out) if provided, pHint will be filled with a non-authoratative result for the privilege check. Unless the return value is zero (0), in which case, this value can be considered autorotative.
*pUiMessage- this parameter should be non-NULL for the UI functionality to be enabled, otherwise, no UI is displayed. However, the value of the string is not used on PS4.
Returns
int32_t - <0 = error, 0 = success, use *pHint for the result, >0 = the request id to pass into PrivilegeApiCheckResult()
Version
09/02/2013 (mcorcoran)
