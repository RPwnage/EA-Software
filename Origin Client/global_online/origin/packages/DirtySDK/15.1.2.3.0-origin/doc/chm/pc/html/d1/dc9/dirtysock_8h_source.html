47 /*!
48  \Mainpage DirtySock
49 
50  \Section intro Introduction
51 
52  DirtySock is a collection of platform-independent APIs providing access to common
53  network library functions. At the most fundamental base level, it provides a Sockets
54  interface similar in many respects to BSD/Sockets and Winsock but with much of the
55  incongruities of those APIs addressed. To support the upper layer protocols, the
56  DirtySock library also includes some basic systems functions like memory management,
57  critical sections, idle functions, and diagnostic output.
58 
59  Running on top of DirtySock Sockets are a number of different protocols. These protocols
60  include both industry standards such as SSL and HTTP/HTTPS, as well custom protocols that
61  implement more game-specific functionalies such as reliable UDP and controller passing.
62  For more on protocols, follow the Comm/Game and Proto links below.
63 
64  Also included in the DirtySock SDK are encryption routines used by the secure protocols,
65  utility functions for processing data such as images, Base64 data, XML, and JSON.
66 
67  Current platforms supported by DirtySock include: Android, iOS, OSX, OSX64, Linux,
68  PC, PC64, PS4, WinPRT and Xbox One.
69 
70  \Section overview Overview
71 
72  DirtySock can be broken down into several basic categories:
73 
74  - \link Comm Comm \endlink - Low-level UDP communications protocols, these are accessed via the NetGame interface
75  - \link Crypt Crypt \endlink - Cryptography routines, implementing industry-standard hash and encryption algorithms
76  - \link DirtySock DirtySock \endlink - Platform-independent sockets layer
77  - \link Game Game \endlink - Game networking modules, utilizing Comm protocols, VoIP, and NAT technologies
78  - \link Graph Graph \endlink - Image decoding routines covering JPEG, GIF, and PNG formats
79  - \link Misc Misc \endlink - Miscelleneous APIs; other network APIs
80  - \link User User \endlink - Miscelleneous APIs; other network APIs
81  - \link Proto Proto \endlink - Implementations of mostly industry-standard protocols, with a few custom ones
82  - \link Util Util \endlink - Utility functions generally involving data parsing, processing, and conversion of industry-standard data formats
83  - \link Voip Voip \endlink - Platform-independent VoIP functionality (not available on all platforms)
