Detailed Description

This module implements an HTTP server that can perform basic transactions (get/put) with an HTTP client. It conforms to but does not fully implement the 1.1 HTTP spec (http://www.w3.org/Protocols/rfc2616/rfc2616.html), and allows for secure HTTP transactions as well as insecure transactions.
