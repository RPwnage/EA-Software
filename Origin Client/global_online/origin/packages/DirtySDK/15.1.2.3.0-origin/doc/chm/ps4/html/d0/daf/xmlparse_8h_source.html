8  This is a simple XML parser for use in controlled situations. It should not

10  a particular host before using it). This only implement a small subset of full
11  XML and while suitable for many applications, it is easily confused by badly
12  formed XML or by some of the ligitimate but convoluated XML conventions.

14  In particular, this parser cannot handle degenerate empty elements (i.e., <BR><BR>

16  predefined entity types (&lt; &gt; &amp; &apos; &quot) are supported. This module
