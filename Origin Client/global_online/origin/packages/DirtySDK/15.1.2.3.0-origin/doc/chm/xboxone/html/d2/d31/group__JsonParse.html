Locate a JSON element.

Parameters

*pName- name of element to locate
Returns
const char * - pointer to element, or NULL if not found

A name can include a period character '.' which indicates an object reference, or an open bracket '[' which indicates an array reference. A bracket must be the last character in the name.

int32_t  iMax,
const char *  _pSrc,
int32_t  iLen

Pre-parse a JSON buffer for fast parsing. Must be called on a buffer before any Find() or Get() functions are used. Pass pDst=NULL and iMax=0 to query the required table size.

Parameters
*pDst- [out] storage for lookup buffer, two elements per object
iMax- number of uint16_t elements in output buffer
*_pSrc- pointer to JSON buffer
iLen- length of JSON buffer; if -1 a strlen is performed
Returns
int32_t - zero=table too small, else size of table in bytes
