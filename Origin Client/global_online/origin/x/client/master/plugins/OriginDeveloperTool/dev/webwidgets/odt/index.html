General Overrides

These overrides affect all of Origin and are not game specific.
For More details visit the Origin Publishing documentation.

Store Environment This override is used to test the different store environments within the Origin client.

Geo IP Used by the Origin Storefront and Downloads. Simulates Origin experience in any supported country. It does not affect the Client Language. Additional IP addresses can be added for countries not listed.
For More details visit the Origin Publishing documentation.

Please note that your account must be entitled to the Origin Developer Tool offer to override Geo-IP.
Please Enter an IP Address

Origin Geo Test The Origin Storefront does not respect the above Geo IP setting by default. If wish to simulate the Origin Storefront as if you were in another country, please ask your store contact for an access code and enter it here.

Environment Enables Origin to use different testing environments. The list is pre-populated with the two common testing environments and enables users to setup additional environments.
For More details visit the Origin Publishing documentation.

Please Enter an Environment

Download CDN Select between Akamai and Level 3. Selecting Akamai will cause Origin to download game files from the Akamai CDN, and selecting Level 3 will cause Origin to download game files from the Level 3 CDN.

Promo Manager This override is used to turn off Promo Manager prompts.

Message of the Day Marking this checkbox will prevent the Message of the Day Dialog from appearing.

Web Cache Flush Marking this checkbox will clear the web cache every time origin starts.

Download Debugging Marking this checkbox will allow you to see file-specific information when downloading or updating a title.

Web Debugging Marking this checkbox will allow you to view debug information for most web views.

Purge Licenses Marking this checkbox will purge all OOA licenses every time Origin starts.

R&D Mode Marking this checkbox disables Origin R&D mode, allowing game information to display in various social features.

Note: Be mindful of using this flag when testing unreleased content or development tools!

Net Promoter Marking this checkbox will cause the Net Promoter dialog to show every time the user logs in.

Telemetry Debugging Marking this checkbox will cause all telemetry events to be duplicated in a local XML file and allows you to access the Telemetry Live Viewer from the Utilities tab.

Debug Menu Marking this checkbox will make advanced debugging options visible in the client's Debug menu.

Install and Launch Overrides

Installation Location Registry Key Used for testing initial setup and install configuration.
For EA Installer titles, this value should be [HKEY_LOCAL_MACHINE\SOFTWARE\<Studio>\<Generic Title Name>\Install Dir]
For more details visit the Origin Publishing documentation.

Installer Name Used for initial setup and install troubleshooting. This is not needed for most override testing. Enter the name of the file that Origin should check for to consider this game installed. Include the relative path. Example: "\bin\bf3.exe"
For More details visit the Origin Publishing documentation.

Game Executable Name Used for initial setup and install troubleshooting. This is not needed for most override testing. Enter the name of the game executable that Origin should launch. Include the relative path. Example: "\bin\bf3.exe"
For More details visit the Origin Publishing documentation.

Companion Website Overrides

Game Launcher URL Used for games with companion websites such as Battlelog. This can be configured with the following placeholder tokens:
• {authcode} – Origin will replace this token with an ID2 authcode for the current user.
• {locale} – Origin will replace this token with the locale that the game is installed in. Example values include "en_US" and "pt_PT".

Game Launcher URL (Integration) This optional override is identical to Game Launcher URL, but will be used when the client is authenticated against Integration Nucleus.

Game Launcher URL Client ID If your companion website utilizes SSO, you must specify an ID2 client_id. This will be used to retrieve an ID2 authcode if you are using the {authcode} token in the Game Launcher URL.

Game Launcher Software Id Override Used for games with launcher programs such as Battlefield. Insert the software Id of the title to be launched instead of this title.

Achievement Overrides

Achievement Set Override Attach an Achievement Set to this offer.
An Achievement Set Id is the concatenation of FranchiseId_MasterTitleId[_PlatformId] - where PlatformId is optional if achievements are shared among all platforms.
Click 'Generate' to automatically create this default Id, or manually specify a custom Id for testing.
