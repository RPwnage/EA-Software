This element is used to describe status comment.

## Usage ##

Enter the element in the Manifest file and and enter comment text as an inner text.


{{% panel theme = "info" header = "Note:" %}}
HTML enabled
{{% /panel %}}
## Example ##


```xml
<statusComment>Release</statusComment>.
```
